Convert-Sturing
===============

A simple program that can decode your *.mt files to *.t .

# Simple *.mt file to convert :

---
```
   0A->1AR  comment
   1A->0AR

```
---

# Output file ( *.t ):

---
```
   A: 
   0 1 R A
   1 0 R A
```
---
